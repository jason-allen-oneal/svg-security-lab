🚀 Quick Start

Getting Started with SVG Security Lab

  1. Start the server: python tools/server.py
  2. Open your browser: Navigate to http://localhost:8000
  3. Begin with attacks: Visit the vulnerable demo to see how SVG attacks work
  4. Learn defenses: Explore CSP, DOMPurify, and Trusted Types demos
  5. Practice detection: Use the scanning tools to analyze SVG files

🟢 Defense Mechanisms

Learn how to protect against SVG-based attacks using modern security technologies:

🛡️

Content Security Policy

Block script execution and restrict resource loading with CSP headers.

 🧹

DOMPurify Sanitization

Clean SVG content by removing dangerous elements and attributes.

 🔐

Trusted Types

Prevent DOM-based XSS by enforcing type safety for dangerous sinks.
