Trusted Types Protection

✅ Trusted Types Active: Trusted Types policy is configured and protecting against DOM-based XSS

Trusted Types is a web platform API that helps prevent DOM-based XSS attacks by requiring developers to sanitize values before they can be assigned to DOM properties that can execute code.

How Trusted Types Works:

  • Policy Creation: Define policies for sanitizing untrusted content
  • Type Enforcement: Browser enforces that only Trusted Types can be assigned to dangerous sinks
  • Sanitization: Untrusted content must be processed through a policy before use

Trusted Types Implementation

1. Basic Trusted Types Setup

// Enable Trusted Types enforcement if (window.trustedTypes && window.trustedTypes.createPolicy) { // Create a default policy window.trustedTypes.createPolicy('default', { createHTML: (string) => string.replace(/<script>/gi, ''), createScript: (string) => string, createScriptURL: (string) => string }); }

2. Custom SVG Sanitization Policy

// Create a policy specifically for SVG content const svgPolicy = trustedTypes.createPolicy('svg-policy', { createHTML: (string) => { // Remove script tags and event handlers return string .replace(/<script[^>]*>.*?<\/script>/gi, '') .replace(/on\w+\s*=\s*["'][^"']*["']/gi, '') .replace(/<foreignObject[^>]*>.*?<\/foreignObject>/gi, ''); } });

3. Using Trusted Types with innerHTML

// This will be blocked without Trusted Types element.innerHTML = untrustedSVG; // ❌ Blocked // This works with Trusted Types element.innerHTML = svgPolicy.createHTML(untrustrustedSVG); // ✅ Allowed

4. Content Security Policy Integration

// CSP header to enable Trusted Types Content-Security-Policy: require-trusted-types-for 'script'; trusted-types 'svg-policy' 'default';

Attack Prevention Examples

Attack Vector 1: Script Injection

// Malicious input const maliciousSVG = "<svg><script>alert('XSS')</script></svg>"; // Without Trusted Types (vulnerable) element.innerHTML = maliciousSVG; // ❌ Executes script // With Trusted Types (protected) element.innerHTML = svgPolicy.createHTML(maliciousSVG); // ✅ Script removed

Attack Vector 2: Event Handler Injection

// Malicious input with event handler const maliciousSVG = "<svg><rect onload='alert(\"XSS\")'/></svg>"; // Trusted Types sanitization removes event handlers element.innerHTML = svgPolicy.createHTML(maliciousSVG); // ✅ Event handler removed

Attack Vector 3: Foreign Object Injection

// Malicious input with foreignObject const maliciousSVG = "<svg><foreignObject><script>alert('XSS')</script></foreignObject></svg>"; // Trusted Types sanitization removes foreignObject element.innerHTML = svgPolicy.createHTML(maliciousSVG); // ✅ foreignObject removed
